Project Description

This homepage acts as a central hub for prospective collaborators, recruiters, and classmates to understand my cybersecurity journey.

The experience opens with an expressive hero section, then guides visitors toward work samples, resume details, and connection points. While the look embraces a technical aesthetic, the content aims to stay personable and welcoming.

  • Highlight professional milestones alongside ongoing graduate coursework.
  • Make it easy to jump to resume, portfolio projects, and contact links.
  • Showcase creative coding experiments that reflect my curiosity beyond security engineering.

User Personas

Three primary audiences inform the tone, structure, and calls to action across the homepage.

Amelia Chen — Security Hiring Manager

  • Goals: Quickly evaluate my technical breadth and communication style.
  • Needs: A concise snapshot plus links to deeper project write ups.
  • Pain points: Limited interview bandwidth, so clarity up front is vital.

Diego Ruiz — Graduate Classmate

  • Goals: Spot shared interests for upcoming course projects or hackathons.
  • Needs: Proof of collaboration tools, availability, and focus areas.
  • Pain points: Hard to gauge team fit when classmates' work is scattered across platforms.

Lina Patel — Creative Technologist

  • Goals: Find inspiration in the crossover between artful UI and rigorous engineering.
  • Needs: Indicators that my experiments balance playfulness with technical depth.
  • Pain points: Many portfolios feel interchangeable and lack a personal narrative.

User Stories

Each persona informs a key interaction that the homepage must make effortless.

Hiring Manager

As a hiring manager, I want to skim a summary of Kush's recent roles and skills so that I can decide whether to schedule an interview.

Classmate

As a classmate, I want to see collaboration interests and current coursework so that I can pitch project ideas that align with Kush's goals.

Creative Technologist

As a fellow creator, I want to explore interactive experiments so that I can draw inspiration and offer feedback for future iterations.

Design Mockup

The mockup balances a bold hero treatment with modular content blocks that can grow alongside future projects.
